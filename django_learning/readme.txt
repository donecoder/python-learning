这是一个学习django的工程。


必要环境:
Django==2+
安装xadmin：
pip install https://codeload.github.com/sshwsfc/xadmin/zip/django2

在pycharm中，环境配置如下：
1、在File | Settings | Project: python-learning | Project Interpreter 设置好python环境.
2、设置Django环境，File | Settings | Languages & Frameworks | Django中
enable Django Support启用
Django project root:   E:\workspace-python\python-learning\django-learning
settings :  django-learning\settings.py

运行命令：python manage.py runserver
